This plugin adds a new condacto that shows a list of the saved games in the browser session and add import/export capabilities.

Example:

 GAMES _
  LISTSAVEDGAMES
  DONE

Use:
- Copy listsavedgames.jsp to the games folder /jsl
- Add an entry to the .txp using the condact LISTSAVEDGAMES
- Compile
